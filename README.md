# sonar-jenkins-intigration-pipeline
**prerequisite**
step1: jenkins need to be installed (https://devops4solutions.medium.com/jenkins-continuous-integration-8c327a9e005c )
step2: sonarqube is need to install in same or diff server (https://devops4solutions.com/sonarqube-installation-windows/) or( https://medium.com/@amitvermaa93/sonarqube-setup-windows-e6a6c01be025 )
step3: install sonarqubescanner in sonarqube server (downlod from link https://docs.sonarqube.org/latest/analysis/scan/sonarscanner/ and un zip it)
step3: in jenkins  need to be configure
       i)  java
       ii) GIT
       iii) extended email notification , emailnotification(https://medium.com/@samiksha.ojha/configuring-email-notifications-in-jenkins-dd745e8270ef#:~:text=Go%20to%20Jenkins%20homepage%20Click,Actions%20%3E%20Select%20editable%20email%20notifications.)
       iv) sonarqube (https://medium.com/@amitvermaa93/jenkins-sonarqube-integration-129f5c49c4ca )
step4: in sonarqube server we have to setup email notification.(https://medium.com/@amitvermaa93/sonarqube-report-on-e-mail-f7764e20b38a_)
step5: write jenkins file and run it from jenkins
